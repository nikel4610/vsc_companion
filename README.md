# Codex Companion

VSCode 안에서 작업 습관을 챙겨주는 개인용 컴패니언 확장 프로그램입니다.

## 목표

- 사이드바 Webview에 컴패니언 UI 표시
- Git 변경 파일 수와 작업 상태를 조용히 알려주기
- 커스텀 명령 버튼으로 자주 쓰는 프로젝트 명령 실행
- 캐릭터 이미지와 메시지 톤을 설정으로 교체 가능하게 만들기

## 개발

```bash
npm install
npm run compile
npm run watch
```

VSCode에서 `F5`를 눌러 Extension Development Host를 실행합니다.
